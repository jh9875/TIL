### Garbage Collection이란?

애플리케이션 실행 중 더이상 사용하지 않는 동적으로 할당된 메모리에 대해 자원을 회수하는 것을 말한다.

자바 애플리케이션을 실행시키면 JVM에서 Garbage Collection 작업을 수행한다.

C나 C++과 달리 JVM에서 직접 메모리 관리를 해주므로 개발자가 직접 메모리를 관리할 필요가 없다.
